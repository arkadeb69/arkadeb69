### Featured Projects

#### [Resume X AI](https://github.com/arkadeb69/resume-x-ai)
An AI-powered resume analysis engine that parses candidate resumes against targeted job roles, instantly identifies skill gaps, and generates tailored learning roadmaps.
* **Core Stack:** Node.js, Express, Google Gemini API, Resend API, Tesseract.js (Client-Side OCR), pdf.js
* **Technical Highlight:** Offloads OCR processing to the browser using `pdf.js` and `Tesseract.js` for zero-server document parsing. Utilizes Gemini's strict JSON schema enforcement to return deterministic analysis structures.

#### [CGEC Capture Crew Portal](https://github.com/arkadeb69/CGEC_Capture_Crew)
The official photography club portal for Cooch Behar Government Engineering College. Serves as a digital gallery, recruitment dashboard, and certificate verification center.
* **Core Stack:** React, Vite, Firebase (Auth/Firestore/Storage), Resend API, Vercel Serverless
* **Technical Highlight:** Implements custom Vercel Edge functions for dynamic OpenGraph/SEO meta injection (intercepting requests and querying Firestore REST endpoints) to support rich social previews.

#### [Credit Bill Manage](https://github.com/arkadeb69/Credit-Bill-manage)
A high-fidelity, offline-first client-side credit bill management portal designed for personal finance tracking.
* **Core Stack:** React 19, Vite, HTML5, CSS3, Browser LocalStorage API
* **Technical Highlight:** Built with a deterministic calculation engine that automatically calculates card status tags (Paid, Pending, Partial) and runs a custom mathematical aging system for warning thresholds.

#### [PC Remote Lock (win-remote-toolkit)](https://github.com/arkadeb69/win-remote-toolkit)
A mobile-friendly Flask web application that allows you to lock, sleep, or shutdown your Windows PC from any device on your local network or securely from anywhere over the internet.
* **Core Stack:** Python, Flask, Windows API (win32-api), Cloudflare Tunnels
* **Technical Highlight:** Triggers system actions via native Windows OS APIs. Securely exposed through a windowless background daemon (VBS runner) and a Cloudflare Tunnel router, complete with PBKDF2/scrypt password hashing protection.
